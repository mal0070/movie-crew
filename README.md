# Movie Crew

## 영화의 꿈을 가진 모두가 꿈을 이룰 수 있도록

무비크루 팀은 막 영화계에 발을 들이는 초보자들을 위해, 필요한 정보를 한 곳에 모았습니다.

배포주소: https://movie-crew.vercel.app/

<br/>

## 기술 스택

| 카테고리          | 사용 기술                         |
| ----------------- | --------------------------------- |
| **라이브러리**    | React + Vite                         |
| **언어**          | Javascript                        |
| **스타일링** | Tailwind CSS                            |
| **API/스토어**    | Google Sheets API   |
| **인증**          | Firebase                          |
| **배포**          | Vercel                            |

<br/>



## 🔍 우리가 발견한 문제

**영화 인력 구직·구인의 현실..**

- 구직자는 여러 플랫폼을 돌아다니며 신뢰 안 가는 글을 뒤적입니다.
- 영화 작업할 시간도 부족한데, 정보를 찾는 데에도 **너무 많은 시간과 체력**이 듭니다.
- 그러다 결국 좋은 기회를 놓치거나, **사기 피해**로 이어지기도 합니다.
